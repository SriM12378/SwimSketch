# SwimSketch

A swim practice planning tool built as a single-page web app. Coaches can schedule timed practice sessions, drag exercises onto lane timelines, assign swimmers to lanes, and save reusable presets.

## How to Run

No build step or server required. Just open the file in a browser:

```
open swimsketch_app.html
```

Or double-click `swimsketch_app.html` in Finder. It runs entirely in the browser — no dependencies to install, no backend needed.
